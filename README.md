# 黄金矿工 - HTML5游戏

一个基于HTML5 Canvas的经典黄金矿工游戏，支持PC和移动端。

## 游戏特色

- 🎣 **经典玩法**：发射钩子抓取地下宝物
- 💎 **多种物品**：金块、钻石、石头等不同价值的物品
- ✨ **炫酷特效**：粒子爆炸、分数弹出、抓取特效
- 📱 **移动端适配**：完美支持触摸操作
- 🎮 **关卡系统**：逐级递增的挑战难度
- ⏱️ **限时挑战**：每关60秒时间限制

## 操作方式

### PC端
- **空格键**：发射钩子
- **点击按钮**：开始游戏、暂停、发射钩子

### 移动端
- **触摸🎣按钮**：发射钩子
- **触摸控制按钮**：开始游戏、暂停

## 技术特点

- **纯前端实现**：HTML5 + CSS3 + JavaScript
- **响应式设计**：适配各种屏幕尺寸
- **模块化架构**：分离的HTML、CSS、JavaScript文件
- **流畅动画**：requestAnimationFrame优化的游戏循环

## 游戏规则

1. 在60秒内达到目标分数即可进入下一关
2. 不同物品有不同价值和重量
3. 抓取重物时钩子收回速度会变慢
4. 时间结束未达标则游戏失败
